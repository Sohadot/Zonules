Active Inference

Active inference frames all perception as interpretation

Definition

Active inference is the framework in which organisms minimize surprise by both updating internal models (perception) and acting on the environment to match predictions.

Scientific Grounding

Proposed by Karl Friston, the framework unifies perception, action, and learning under a single objective: the minimization of free energy or surprisal. Active inference generalizes predictive coding by including action as a mechanism for resolving prediction error, not merely updating beliefs.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Active inference frames all perception as interpretation (FIO-05): the organism does not passively receive the world but actively constructs it, and its actions are determined by its generative model.

Relationship to the Governing Thesis

Active Inference is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-05 — Interpretation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.

Mechanism

Active inference, developed from the free-energy principle, treats perception and action as two routes to one goal: reducing the discrepancy between what the system predicts and what it senses. The key move beyond pure predictive coding is that prediction error can be resolved in two ways — the organism can update its internal model to fit the input (perception), or it can act on the world to make the input fit the model (action). Moving the eyes to fixate a target is, in this account, as much a part of inference as revising a belief about it.

Failure Mode

Because the model is maintained by continuous action, its characteristic failure is not a missing signal but a misalignment that action fails to close. The system stays confident in a model the world no longer supports and does not sample the evidence that would correct it. In focus-integrity terms this is an Interpretation Failure (FIO-05) that persists precisely because the loop meant to fix it is not doing so.

Human–Machine Bridge

Active inference is influential in part because it describes agents in general, not only brains. An autonomous machine perception system likewise maintains a model and acts — choosing where to point a sensor, what to attend to next — to reduce its own uncertainty. Reading the framework through focus integrity makes the parallel concrete: the same active loop that holds a human percept steady is what a machine must implement to keep its interpretation aligned with a changing scene.
