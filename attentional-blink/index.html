Attentional Blink

The attentional blink is an accommodation failure

Definition

The attentional blink is the temporary impairment in detecting a second target that occurs when it follows a first target within roughly 200-500 ms in a rapid serial visual presentation.

Scientific Grounding

The phenomenon is attributed to the consolidation of the first target into working memory, which temporarily depletes attentional resources for subsequent items. The attentional blink can be reduced or eliminated if the second target immediately follows the first (Lag-1 sparing effect) or if task demands are reduced.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

The attentional blink is an accommodation failure (FIO-02): the system temporarily cannot extend its processing range to a second target, creating a temporal gap in the perceptual record.

Relationship to the Governing Thesis

Attentional Blink is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-02 — Accommodation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
