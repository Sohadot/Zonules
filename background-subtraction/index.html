Background Subtraction

The moving signal pulled from a static surround.

Definition

Background subtraction separates moving foreground objects from a static background in video.

Scientific Grounding

It maintains a model of the background scene and flags the pixels that deviate from it. Changing illumination and moving backgrounds are classic sources of error.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Background subtraction is a separation-layer operation: it pulls the relevant moving signal out of a static surround.
