The Binding Problem

Coherent objects must be assembled from distributed signals.

Definition

The binding problem concerns how the visual system combines separately processed features, such as color, shape, and motion, into unified object representations.

Scientific Grounding

Attention is thought to play a role in correctly binding features to objects. Failures of binding can produce illusory conjunctions, where features are mis-combined.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

The binding problem is an interpretation challenge (FIO-05): coherent meaning must be assembled from distributed signals.

Relationship to the Governing Thesis

The Binding Problem is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-05 — Interpretation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
