Camera Fingerprinting

An image tied back to the device that made it.

Definition

Camera fingerprinting identifies the source device of an image from sensor-specific noise patterns.

Scientific Grounding

The photo-response non-uniformity of a sensor acts as a stable, device-unique signature. A mismatch between a region's fingerprint and the rest of an image can reveal tampering.

*Sources: SRC-023, SRC-024. See the source registry for classification.*

Machine-Vision Role

Camera fingerprinting is a provenance-layer trace: it ties an image back to the device that captured it.
