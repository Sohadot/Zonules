Cognitive Load

Cognitive load is an accommodation constraint

Definition

Cognitive load is the total amount of mental effort being used in working memory at a given time, limiting the capacity available for perception and task performance.

Scientific Grounding

Working memory capacity is limited to roughly four chunks of information simultaneously; tasks that exceed this limit cause performance degradation and errors. High cognitive load reduces perceptual sensitivity, increasing inattentional blindness rates and narrowing the focus of attention.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Cognitive load is an accommodation constraint (FIO-02): it defines the operating envelope of the perceptual system under demand, narrowing the range of what can be held in focus when resources are saturated.

Relationship to the Governing Thesis

Cognitive Load is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-02 — Accommodation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
