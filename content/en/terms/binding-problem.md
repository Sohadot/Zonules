# The Binding Problem

> Coherent objects must be assembled from distributed signals.

## Definition

The binding problem concerns how the visual system combines separately processed features, such as color, shape, and motion, into unified object representations. [CLM-296]

## Scientific Grounding

Attention is thought to play a role in correctly binding features to objects. [CLM-297] Failures of binding can produce illusory conjunctions, where features are mis-combined. [CLM-298]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

The binding problem is an interpretation challenge (FIO-05): coherent meaning must be assembled from distributed signals. [CLM-299]
