# Bottom-Up Processing

> Bottom-up processing is a signal-reading stage

## Definition

Bottom-up processing refers to perceptual analysis driven by the properties of the sensory input itself, proceeding from local features to global interpretation without prior knowledge guiding the analysis. [CLM-512]

## Scientific Grounding

Low-level features such as edges, orientations, and color contrasts are extracted in early visual cortex before categorical knowledge is applied. [CLM-513] Bottom-up processing is captured in computational models of saliency, which predict which regions attract attention based on local feature contrast alone. [CLM-514]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Bottom-up processing is a signal-reading stage (FIO-03): it extracts the features of the signal without prediction, providing the raw data on which top-down interpretation subsequently operates. [CLM-515]
