# Cognitive Load

> Cognitive load is an accommodation constraint

## Definition

Cognitive load is the total amount of mental effort being used in working memory at a given time, limiting the capacity available for perception and task performance. [CLM-516]

## Scientific Grounding

Working memory capacity is limited to roughly four chunks of information simultaneously; tasks that exceed this limit cause performance degradation and errors. [CLM-517] High cognitive load reduces perceptual sensitivity, increasing inattentional blindness rates and narrowing the focus of attention. [CLM-518]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Cognitive load is an accommodation constraint (FIO-02): it defines the operating envelope of the perceptual system under demand, narrowing the range of what can be held in focus when resources are saturated. [CLM-519]
