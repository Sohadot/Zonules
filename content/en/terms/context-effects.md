# Context Effects

> The same mark, a letter here and a number there.

## Definition

Context effects are the influence of surrounding information on how an ambiguous stimulus is perceived. [CLM-744]

## Scientific Grounding

The same shape can be read as a letter or a number depending on the items flanking it. [CLM-745] Scene context speeds recognition of objects that are consistent with it and slows recognition of inconsistent ones. [CLM-746]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Context effects are an interpretation-layer bias — surrounding evidence steers how the signal is read. [CLM-747]
