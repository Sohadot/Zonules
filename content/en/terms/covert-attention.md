# Covert Attention

> Focus held on a target the eye does not point at.

## Definition

Covert attention is the selection of a location or object without moving the eyes to it. [CLM-1044]

## Scientific Grounding

Attention can be shifted covertly to the periphery, improving detection there while fixation stays fixed. [CLM-1045] Covert shifts typically precede and help guide subsequent eye movements. [CLM-1046]

*Sources: SRC-005, SRC-009. See the source registry for classification.*

## Perceptual Role

Covert attention is a suspension-layer act: it holds focus on a target the eye is not pointed at. [CLM-1047]
