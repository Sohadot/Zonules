# Feature-Based Attention

> Feature-based attention is a signal selection mechanism

## Definition

Feature-based attention is the selective enhancement of processing for a particular visual feature value — a color, orientation, or motion direction — across the entire visual field simultaneously. [CLM-536]

## Scientific Grounding

Attention to a feature value such as red enhances processing of all red items in the scene regardless of location and suppresses processing of opposite feature values. [CLM-537] Feature-based attention operates globally and simultaneously across the visual field, in contrast to spatial attention, which is local and sequential. [CLM-538]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Feature-based attention is a signal selection mechanism (FIO-03): it amplifies signal at a target feature value while suppressing flanking values, reducing the signal-noise ratio for that feature across the scene. [CLM-539]
