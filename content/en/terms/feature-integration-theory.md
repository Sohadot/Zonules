# Feature Integration Theory

> Attention is the glue that binds features into objects.

## Definition

Feature integration theory proposes that simple features such as color and orientation are registered in parallel across the visual field, while combining them into objects requires focused attention. [CLM-704]

## Scientific Grounding

It explains why searching for a single feature is fast and parallel, whereas searching for a conjunction of features is slow and serial. [CLM-705] Without attention, features can be miscombined into illusory conjunctions. [CLM-706]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Feature integration theory frames binding as an interpretation-layer act — attention is what turns separate features into a correctly read object. [CLM-707]
