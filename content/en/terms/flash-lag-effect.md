# Flash-Lag Effect

> The system reads where a moving thing should be.

## Definition

The flash-lag effect is the illusion that a moving object is ahead of a flash presented at the same physical location. [CLM-1004]

## Scientific Grounding

It suggests the visual system extrapolates the position of moving objects to offset neural delays. [CLM-1005] The size of the effect depends on the speed of the moving object. [CLM-1006]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

The flash-lag effect is an interpretation-layer correction: the system reads where a moving thing should be, not where it was. [CLM-1007]
