# Holistic Processing

> A face is read whole, not feature by feature.

## Definition

Holistic processing is the perception of a stimulus as an integrated whole rather than a set of independent parts, most studied for faces. [CLM-740]

## Scientific Grounding

The composite and part-whole effects show that face features are hard to judge in isolation from the whole face. [CLM-741] Inverting a face disrupts holistic processing and sharply impairs recognition. [CLM-742]

*Sources: SRC-015, SRC-005. See the source registry for classification.*

## Perceptual Role

Holistic processing is an interpretation-layer mode — meaning read from the configuration, not the catalog of parts. [CLM-743]
