# Iconic Memory

> A vivid afterimage of the scene, gone in a heartbeat.

## Definition

Iconic memory is a very brief, high-capacity visual store that holds a detailed image of a scene for a fraction of a second after it disappears. [CLM-696]

## Scientific Grounding

Partial-report experiments show that far more is briefly available than can be reported before the trace fades. [CLM-697] The icon decays within roughly a few hundred milliseconds unless its contents are transferred to a more durable store. [CLM-698]

*Sources: SRC-005, SRC-009. See the source registry for classification.*

## Perceptual Role

Iconic memory is a suspension-layer buffer — it holds the visual signal briefly so later stages can act on it. [CLM-699]
