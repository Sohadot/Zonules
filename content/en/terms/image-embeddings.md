# Image Embeddings

> Each image held as a point in a reasoned-over space.

## Definition

An image embedding is a compact vector that represents an image so that similar images lie close together. [CLM-1192]

## Scientific Grounding

Embeddings let images be compared, searched, and clustered by vector distance. [CLM-1193] Learned embeddings underpin retrieval, recognition, and cross-modal matching with text. [CLM-1194]

*Sources: SRC-017, SRC-022, SRC-018. See the source registry for classification.*

## Machine-Vision Role

Image embeddings are a suspension-layer coordinate: they hold each image as a point in a space the system can reason over. [CLM-1195]
