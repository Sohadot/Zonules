# Pattern Recognition

> Recognition is a match against memory, and the match can quietly impose meaning the signal never carried.

## Definition

Pattern recognition is a cognitive process by which the visual system identifies stimuli by matching sensory input against stored memory templates or prototypes. [CLM-160]

## Scientific Grounding

Gestalt principles — proximity, similarity, continuity, closure — describe the brain's automatic tendencies to group visual elements into coherent patterns. [CLM-161] Pattern recognition operates at multiple processing levels simultaneously, including low-level feature detection and high-level categorical recognition. [CLM-162]

*Sources: SRC-005, SRC-006. See the source registry for classification.*
