# Perceptual Adaptation

> Perceptual adaptation is a recalibration of interpretation

## Definition

Perceptual adaptation is the process by which the visual system recalibrates its sensitivity or interpretation framework following prolonged exposure to a particular stimulus. [CLM-524]

## Scientific Grounding

After prolonged exposure to a tilted grating, a vertical grating appears tilted in the opposite direction — the tilt after-effect — demonstrating that adaptation alters the perceptual reference for orientation. [CLM-525] Adaptation operates at multiple levels of the visual system, from early photoreceptor adaptation to high-level categorical after-effects on face and object perception. [CLM-526]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Perceptual adaptation is a recalibration of interpretation (FIO-05): the visual system adjusts the reference frame it uses to assign meaning to incoming signals, making the same signal produce a different output after exposure. [CLM-527]
