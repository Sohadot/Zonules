# Preattentive Processing

> The instant, parallel sweep before focus arrives.

## Definition

Preattentive processing is the rapid, parallel analysis of basic visual features across the whole field before focused attention is applied. [CLM-708]

## Scientific Grounding

Features that pop out, such as a single red item among green ones, are detected in roughly constant time regardless of the number of distractors. [CLM-709] It provides the salience map that guides where attention is subsequently directed. [CLM-710]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Preattentive processing is a separation-layer first pass — it segregates candidate signal from background before focus is committed. [CLM-711]
