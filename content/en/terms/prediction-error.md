# Prediction Error

> The part of the input meaning has not yet explained.

## Definition

Prediction error is the mismatch between sensory input and the brain's prediction of that input. [CLM-960]

## Scientific Grounding

In predictive-coding accounts, only this residual error is propagated forward to update the model. [CLM-961] Minimizing prediction error over time is proposed as a core principle of perception. [CLM-962]

*Sources: SRC-021, SRC-020. See the source registry for classification.*

## Perceptual Role

Prediction error is an interpretation-layer signal: it is the part of the input that meaning has not yet accounted for. [CLM-963]

## Relationship to the Governing Thesis

Prediction Error is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the [focus-integrity ontology](/focus-integrity-ontology/) is fixed as an FIO-05 — Interpretation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
