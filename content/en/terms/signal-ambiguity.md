# Signal Ambiguity

> When one image fits two scenes, the signal alone cannot decide which reading is correct.

## Definition

Visual ambiguity occurs when a retinal image is consistent with more than one physical scene configuration, requiring the visual system to resolve it using priors or additional cues. [CLM-184]

## Scientific Grounding

Reversible figures — such as the Necker cube or Rubin vase — are canonical demonstrations of perceptual ambiguity, where the same image alternates between two stable interpretations. [CLM-185] The visual system typically resolves ambiguity by selecting the most probable interpretation given learned priors, but this resolution can be unstable when competing interpretations are equally probable. [CLM-186]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

In the Zonules FIO framework, signal ambiguity is classified under FIO-03 Signal-Noise Separation: it represents the condition where signal definition is insufficient to support a stable, confident read. [CLM-187]
