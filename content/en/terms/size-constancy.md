# Size Constancy

> The same object, the same size, however far it stands.

## Definition

Size constancy is the tendency to perceive an object as having a stable physical size despite changes in the size of its retinal image with distance. [CLM-664]

## Scientific Grounding

The visual system achieves this by scaling the retinal image against perceived distance. [CLM-665] When distance cues are removed or made misleading, size constancy breaks down and produces size illusions. [CLM-666]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Size constancy is an accommodation-layer achievement — the percept holds steady as viewing conditions change. [CLM-667]
