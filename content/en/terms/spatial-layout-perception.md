# Spatial Layout Perception

> The stage on which objects are located.

## Definition

Spatial layout perception is the apprehension of the overall geometric structure of a scene, such as its ground plane and boundaries. [CLM-1032]

## Scientific Grounding

The layout can be grasped rapidly and guides navigation and the placement of objects. [CLM-1033] Surfaces such as the ground plane provide a reference frame for judging object positions and sizes. [CLM-1034]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Spatial layout perception is an interpretation-layer frame: it builds the stage on which objects are located. [CLM-1035]
