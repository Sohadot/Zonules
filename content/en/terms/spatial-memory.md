# Spatial Memory

> Holding where things are when they leave view.

## Definition

Spatial memory is the storage and retrieval of information about the locations of objects and the layout of environments. [CLM-940]

## Scientific Grounding

It supports navigation and the ability to return attention to remembered locations. [CLM-941] It draws on brain regions including the hippocampus and parietal cortex. [CLM-942]

*Sources: SRC-005, SRC-009. See the source registry for classification.*

## Perceptual Role

Spatial memory is a suspension-layer map: it holds where things are when they are no longer in view. [CLM-943]
