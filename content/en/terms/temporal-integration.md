# Temporal Integration

> Temporal integration is an accommodation mechanism

## Definition

Temporal integration is the process by which the visual system combines stimuli presented within a brief time window into a single perceptual event. [CLM-464]

## Scientific Grounding

The temporal resolution of the visual system is limited: stimuli within roughly 50-100 ms of each other may be integrated rather than resolved as separate. [CLM-465] Temporal integration underlies the perception of continuity in flickering light and the inability to resolve rapid sequential events as distinct. [CLM-466]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

Temporal integration is an accommodation mechanism (FIO-02): it extends the operating range of the visual system across time, treating nearby events as continuous when the gap falls below threshold. [CLM-467]
