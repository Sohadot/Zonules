# Vision-Language Models

> Fluent description is not the same as grounded understanding.

## Definition

Vision-language models are machine learning systems that jointly process images and text to perform tasks such as captioning and visual question answering. [CLM-312]

## Scientific Grounding

They learn shared representations that link visual and linguistic information. [CLM-313] Their outputs can be fluent yet ungrounded in the actual content of the image. [CLM-314]

*Sources: SRC-007. See the source registry for classification.*

## Machine-Vision Role

Vision-language models are an interpretation layer (FIO-05): meaning is assigned by linking modalities, where grounding can fail. [CLM-315]
