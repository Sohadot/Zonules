# Visual Inference

> The visual system fills the gaps; the risk is that expectation fills them instead of the signal.

## Definition

The visual system routinely fills in incomplete information, inferring object identity, surface continuity, and depth from partial cues. [CLM-164]

## Scientific Grounding

Visual inference is governed by prior probabilities — the brain combines current sensory evidence with learned expectations to construct a most-likely percept. [CLM-165] Predictive coding models propose that the brain continuously generates predictions about incoming stimuli and only propagates prediction errors, not raw sensory data. [CLM-166]

*Sources: SRC-005, SRC-006. See the source registry for classification.*

## Perceptual Role

In the Zonules FIO framework, visual inference is classified under FIO-05 Interpretation because it introduces the risk that expectations override fidelity to the actual optical signal. [CLM-167]
