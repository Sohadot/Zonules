# Visual Long-Term Memory

> What we keep of an image long after it is gone.

## Definition

Visual long-term memory is the durable store of visual information that can persist for hours, days, or years. [CLM-932]

## Scientific Grounding

Studies show people can remember thousands of images with surprising detail after a single viewing. [CLM-933] Unlike working memory, its capacity appears very large and its contents relatively stable. [CLM-934]

*Sources: SRC-005, SRC-009. See the source registry for classification.*

## Perceptual Role

Visual long-term memory is a suspension-layer store: it holds visual structure available to perception long after the signal is gone. [CLM-935]
