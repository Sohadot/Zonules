# Visual Recognition Memory

> The judgment that a percept is old, not new.

## Definition

Visual recognition memory is the ability to judge whether a currently seen image has been encountered before. [CLM-936]

## Scientific Grounding

Recognition is typically faster and more accurate than free recall of the same material. [CLM-937] It can operate through a sense of familiarity or through explicit recollection of the prior encounter. [CLM-938]

*Sources: SRC-005, SRC-014. See the source registry for classification.*

## Perceptual Role

Visual recognition memory is an interpretation-layer judgment: it reads a current percept as old or new. [CLM-939]
