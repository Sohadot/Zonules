# Working Memory Capacity

> The few items focus can hold at once.

## Definition

Working memory capacity is the limited number of items that can be held in an active, accessible state at once. [CLM-944]

## Scientific Grounding

Visual working memory is typically limited to only a few objects at a time. [CLM-945] Capacity is constrained by both the number and the complexity of the items held. [CLM-946]

*Sources: SRC-005, SRC-014. See the source registry for classification.*

## Perceptual Role

Working memory capacity is a suspension-layer limit: it bounds how much focus can be held available at once. [CLM-947]
