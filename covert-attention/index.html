Covert Attention

Focus held on a target the eye does not point at.

Definition

Covert attention is the selection of a location or object without moving the eyes to it.

Scientific Grounding

Attention can be shifted covertly to the periphery, improving detection there while fixation stays fixed. Covert shifts typically precede and help guide subsequent eye movements.

*Sources: SRC-005, SRC-009. See the source registry for classification.*

Perceptual Role

Covert attention is a suspension-layer act: it holds focus on a target the eye is not pointed at.

Relationship to the Governing Thesis

Covert Attention is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
