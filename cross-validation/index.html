Cross-Validation

Do a model's readings hold on data it did not see?

Definition

Cross-validation estimates how well a model generalizes by repeatedly training and testing on different splits of the data.

Scientific Grounding

It uses the available data more efficiently than a single train-test split. It helps detect overfitting and guides the choice of model and hyperparameters.

*Sources: SRC-027, SRC-017. See the source registry for classification.*

Machine-Vision Role

Cross-validation is an interpretation-layer check: it asks whether a model's readings hold on data it did not see.
