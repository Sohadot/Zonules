Data Augmentation

Widening the range over which a model holds focus.

Definition

Data augmentation is the practice of expanding training data by applying label-preserving transformations to existing images.

Scientific Grounding

Common transformations include cropping, flipping, rotation, and color changes. Augmentation can improve a model's robustness to variation in its inputs.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Data augmentation is an accommodation strategy (FIO-02): it widens the range over which a model holds focus.
