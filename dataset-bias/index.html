Dataset Bias

A model learns the data it is given, flaws and all.

Definition

Dataset bias is the systematic non-representativeness of a training set relative to the conditions where the model will be used.

Scientific Grounding

Models can exploit incidental correlations in the data, such as backgrounds or capture conditions, instead of the intended object. Bias in the data propagates into the model and surfaces as poor or unequal performance on under-represented cases.

*Sources: SRC-022, SRC-017. See the source registry for classification.*

Machine-Vision Role

Dataset bias is an interpretation-layer fault — the model learns to read the wrong evidence and reaches confident wrong conclusions.
