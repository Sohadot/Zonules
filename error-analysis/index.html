Error Analysis

Not how often a model is right, but how it is wrong.

Definition

Error analysis is the systematic study of the cases a model gets wrong.

Scientific Grounding

Grouping errors by type reveals failure patterns that aggregate metrics hide. It guides targeted improvements more effectively than a single accuracy number.

*Sources: SRC-017, SRC-027. See the source registry for classification.*

Machine-Vision Role

Error analysis is an interpretation-layer audit: it asks not how often a model is right but how and why it is wrong.
