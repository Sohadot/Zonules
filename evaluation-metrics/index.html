Evaluation Metrics

What counts as reading the scene correctly.

Definition

Evaluation metrics quantify how well a vision model performs against reference labels.

Scientific Grounding

Different tasks use different metrics, such as accuracy, precision and recall, or intersection-over-union. A metric that misrepresents the goal can make a poor model look strong.

*Sources: SRC-022, SRC-007, SRC-027. See the source registry for classification.*

Machine-Vision Role

Evaluation metrics are an interpretation-layer yardstick: they define what counts as reading the scene correctly.
