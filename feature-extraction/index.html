Feature Extraction

Separating informative signal from raw pixel noise.

Definition

Feature extraction is the process of transforming raw image data into representations that capture informative structure for downstream tasks.

Scientific Grounding

Classical methods use hand-designed descriptors, while modern methods learn features directly from data. The quality of extracted features bounds the performance of downstream recognition.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Feature extraction is a signal-noise step (FIO-03): it separates informative signal from raw pixel noise.
