Feature Integration Theory

Attention is the glue that binds features into objects.

Definition

Feature integration theory proposes that simple features such as color and orientation are registered in parallel across the visual field, while combining them into objects requires focused attention.

Scientific Grounding

It explains why searching for a single feature is fast and parallel, whereas searching for a conjunction of features is slow and serial. Without attention, features can be miscombined into illusory conjunctions.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Feature integration theory frames binding as an interpretation-layer act — attention is what turns separate features into a correctly read object.

Relationship to the Governing Thesis

Feature Integration Theory is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-05 — Interpretation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
