Ces codes sont des Codes d’Intégrité du Focus : FIO désigne la classe de défaillance ; FIS désigne le critère standard correspondant. Que signifient ces codes ?

Authenticité du contenu

Prouver l’origine plutôt que deviner depuis les pixels.

Définition

L’authenticité du contenu désigne les preuves vérifiables de l’origine d’un média et de son historique de modification.

Ancrage scientifique

Les systèmes de provenance attachent des métadonnées signées afin qu’un lecteur puisse vérifier l’origine et la chaîne d’édition.

Rôle dans le système

Elle est FIO-04 : l’intégrité vient d’un mandat externe d’origine que l’analyse visuelle interne ne peut pas fournir seule.

Classification par couche

  • Couche : L3 — Vision machine
  • Classe FIO : FIO-04 — Défaillance de provenance
  • Critère FIS : FIS-4 — Provenance

Mode de défaillance

L’absence de preuve de provenance ne prouve pas la fausseté ; elle doit être lue comme une inconnue.
