Erreur perceptive

Quand un percept clair devient faux.

Définition

Une erreur perceptive apparaît lorsque les heuristiques qui reconstruisent le monde depuis un signal ambigu produisent un percept confiant mais incorrect.

Ancrage scientifique

Les illusions ne sont pas de simples curiosités : elles révèlent les inférences que le système utilise normalement avec succès.

Rôle dans le système

Elle est l’exemple canonique de FIO-05 : le focus est tenu, le signal peut être propre, mais le sens final est erroné.

Classification par couche

  • Couche : L2 — Perception
  • Classe FIO : FIO-05 — Défaillance d’interprétation
  • Critère FIS : FIS-5 — Interprétation

Mode de défaillance

Une erreur perceptive ne prouve pas une mauvaise vue. Elle peut apparaître avec une optique parfaitement fonctionnelle.

Termes connexes
