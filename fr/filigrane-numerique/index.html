Filigrane numérique

Un signal d’origine intégré au média.

Définition

Le filigrane numérique intègre dans une image un signal souvent imperceptible mais récupérable, destiné à porter une information d’origine ou d’authenticité.

Ancrage scientifique

Il peut soutenir l’attribution et la gestion des droits, mais il peut aussi être altéré, supprimé ou contourné selon les transformations.

Rôle dans le système

Il est un outil de provenance : il ajoute une trace que l’apparence seule ne contient pas.

Classification par couche

  • Couche : L3 — Vision machine
  • Classe FIO : FIO-04 — Défaillance de provenance
  • Critère FIS : FIS-4 — Provenance

Mode de défaillance

Un filigrane n’est pas une garantie absolue. Il doit être interprété dans un système de vérification plus large.
