Hachage perceptuel

Un rappel compact de similarité visuelle.

Définition

Le hachage perceptuel résume la structure visuelle d’une image dans un code court, conçu pour rester proche malgré de petites modifications.

Ancrage scientifique

Contrairement à un hachage cryptographique, il accepte la similarité. C’est utile pour retrouver des contenus connus à grande échelle.

Rôle dans le système

Dans la thèse, il donne à la machine une mémoire de provenance : savoir si une image ressemble à une image déjà vue.

Classification par couche

  • Couche : L3 — Vision machine
  • Classe FIO : FIO-04 — Défaillance de provenance
  • Critère FIS : FIS-4 — Provenance

Mode de défaillance

Il ne vérifie pas l’authenticité. Il établit une similarité, avec des risques de faux rapprochement ou d’évasion adversariale.
