Holistic Processing

A face is read whole, not feature by feature.

Definition

Holistic processing is the perception of a stimulus as an integrated whole rather than a set of independent parts, most studied for faces.

Scientific Grounding

The composite and part-whole effects show that face features are hard to judge in isolation from the whole face. Inverting a face disrupts holistic processing and sharply impairs recognition.

*Sources: SRC-015, SRC-005. See the source registry for classification.*

Perceptual Role

Holistic processing is an interpretation-layer mode — meaning read from the configuration, not the catalog of parts.
