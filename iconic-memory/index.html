Iconic Memory

A vivid afterimage of the scene, gone in a heartbeat.

Definition

Iconic memory is a very brief, high-capacity visual store that holds a detailed image of a scene for a fraction of a second after it disappears.

Scientific Grounding

Partial-report experiments show that far more is briefly available than can be reported before the trace fades. The icon decays within roughly a few hundred milliseconds unless its contents are transferred to a more durable store.

*Sources: SRC-005, SRC-009. See the source registry for classification.*

Perceptual Role

Iconic memory is a suspension-layer buffer — it holds the visual signal briefly so later stages can act on it.

Relationship to the Governing Thesis

Iconic Memory is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
