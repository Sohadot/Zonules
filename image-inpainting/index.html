Image Inpainting

It can repair an image or quietly rewrite it.

Definition

Image inpainting fills in missing or removed regions of an image with plausible content.

Scientific Grounding

Learned inpainting models synthesize the missing pixels conditioned on the surrounding context. The same technique that restores damaged photos can seamlessly erase or alter real content.

*Sources: SRC-007, SRC-017, SRC-023. See the source registry for classification.*

Machine-Vision Role

Image inpainting is a provenance-layer tool: it can repair an image or quietly rewrite what it shows.
