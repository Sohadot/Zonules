Multisensory Integration

Multisensory integration is an interpretation

Definition

Multisensory integration is the process by which the brain combines signals from different sensory modalities into a unified percept.

Scientific Grounding

Integration follows the principle of maximum likelihood estimation: signals are combined in proportion to their reliability, with more reliable signals given more weight. Conflicts between modalities are resolved in favor of whichever modality has the highest reliability for that task.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Multisensory integration is an interpretation (FIO-05): the unified percept of an event is a weighted construction from multiple incomplete signals, not a read-out from any single sensory stream.
