Open-Vocabulary Detection

Reading objects it was never trained to name.

Definition

Open-vocabulary detection localizes and names objects from categories not seen during training.

Scientific Grounding

It typically aligns visual regions with a language model's text embeddings to generalize beyond fixed labels. It extends detection from a closed label set toward arbitrary described categories.

*Sources: SRC-007, SRC-019, SRC-017. See the source registry for classification.*

Machine-Vision Role

Open-vocabulary detection is an interpretation-layer reach: it reads objects it was never explicitly trained to name.
