Perceptual Grouping

Perceptual grouping is a signal-noise operation

Definition

Perceptual grouping is the visual system's tendency to organize elements of the visual scene into coherent units based on proximity, similarity, continuity, and common fate.

Scientific Grounding

The Gestalt principles describe the rules of grouping: elements that are close, similar, or moving together are perceived as belonging to the same object or group. Grouping is a pre-attentive process that occurs before focused attention and across the entire visual field simultaneously.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Perceptual grouping is a signal-noise operation (FIO-03): it partitions the incoming signal into figure and ground, separating what belongs to a coherent object from what does not.
