Perceptual Priming

Prior input quietly tilts how the next signal is read.

Definition

Perceptual priming is the facilitation of recognizing a stimulus due to prior exposure to a related stimulus.

Scientific Grounding

Priming can occur without conscious awareness of the prior exposure. It reflects how prior experience shapes subsequent perception.

*Sources: SRC-005. See the source registry for classification.*

Perceptual Role

Perceptual priming is an interpretation bias (FIO-05): prior input tilts how the current signal is read.
