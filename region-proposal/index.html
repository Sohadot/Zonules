Region Proposal

Deciding where the scene is worth reading closely.

Definition

Region proposal generates candidate image regions that may contain objects, for later classification.

Scientific Grounding

It narrows an intractable search over all possible windows to a manageable set of likely regions. Proposal quality bounds the performance of the detection systems built on top of it.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role
