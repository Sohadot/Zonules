Semantic Segmentation

Labelling every pixel is an act of interpretation that can fail silently on anything unfamiliar.

Definition

Semantic segmentation assigns a class label to every pixel in an image, partitioning the image into regions corresponding to recognized categories.

Scientific Grounding

Instance segmentation extends semantic segmentation by distinguishing individual instances of the same class, assigning each pixel both a category label and an instance identity. Semantic segmentation fails silently on out-of-vocabulary regions: pixels belonging to unrecognized categories are assigned to the nearest known class rather than flagged as unknown.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

In the Zonules FIO framework, semantic segmentation is classified under FIO-05 Interpretation: assigning meaning to every pixel is an act of interpretation that can fail systematically without signalling failure.
