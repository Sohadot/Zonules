Neural Network Interpretability

High accuracy is not the same as correct reasoning. Interpretability asks why the machine was right — because if it was right for the wrong reason, it will be wrong when it matters.

Definition

Neural network interpretability is the field concerned with understanding and explaining the internal representations and decision-making processes of deep learning models applied to visual tasks. It seeks to answer: what features does the model use? what image regions influence each prediction? what has the model learned to represent? and whether the model's reasoning is grounded in the visual content that a human observer would identify as task-relevant.

Scientific Grounding

Interpretability methods fall into several categories. Feature visualization generates synthetic inputs that maximally activate individual neurons or channels, revealing what visual patterns each unit responds to. Saliency methods compute gradients or attention weights to generate pixel-level attribution maps indicating which image regions most influenced a given prediction. Probing classifiers train linear models on learned representations to test whether specific properties (shape, colour, texture, object identity) are linearly decodable from intermediate representations.

Gradient-weighted class activation mapping (Grad-CAM) is among the most widely applied attribution methods: it uses the gradient of a class score with respect to convolutional feature maps to highlight image regions that were most influential for the prediction. These visualizations are widely used in medical imaging AI to provide a form of accountability for model decisions — showing which retinal region drove a diabetic retinopathy classification, for example.

Interpretability research has produced a troubling finding: high-accuracy models can rely on spurious correlations in training data, achieving correct outputs for incorrect reasons. A skin lesion classifier was found to be partially using surgical markers (artefacts of the imaging context, not the lesion itself) as a diagnostic cue. A chest X-ray model used patient positioning and scanner manufacturer signatures. These are FIO-05 failures: correct predictions grounded in the wrong features of the signal.

Sources: SRC-007. See source registry.

Interpretability as Focus Integrity Audit

Neural network interpretability is, in Focus Integrity terms, the audit of whether a machine vision system's interpretation is grounded in the signal's actual content. The FIO-05 classification is precise: the failure mode interpretability exists to detect is exactly the condition where the model's output is an interpretation that diverges from the signal — where the model is “right” in a way that cannot be trusted because the grounding is incorrect.

Relationship to the Governing Thesis

Zonules.com treats neural network interpretability as the machine-vision analogue of the FIO-05 class across both L2 and L3. At L2, perceptual error and top-down processing document how human interpretation can diverge from signal content. At L3, interpretability is the methodological toolkit for detecting and documenting the analogous divergence in machine systems. The Focus Integrity Standard requires that for high-stakes machine vision deployments, interpretability analysis be conducted to verify that model predictions are grounded in task-relevant signal features.

Safety Notes

Educational reference only. Interpretability methods do not guarantee model safety or correctness; they are diagnostic tools requiring expert interpretation. This page does not constitute professional or regulatory advice.
