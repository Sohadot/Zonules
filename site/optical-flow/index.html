Optical Flow

Motion is a signal. Optical flow is the method by which that signal is held coherent across time.

Definition

Optical flow is the pattern of apparent motion of image brightness patterns caused by the relative motion between an observer (or camera) and a scene. In machine vision, it is computed from the frame-to-frame differences in a video sequence and expressed as a dense or sparse vector field assigning a velocity estimate to each pixel or tracked feature point.

Scientific Grounding

The foundational constraint of optical flow computation is the brightness constancy assumption: the intensity of a scene point is assumed constant as it moves across frames. Under this assumption, the optical flow constraint equation relates the spatial and temporal image gradients to the flow vector at each point. However, this equation is underdetermined for a single pixel — this is the aperture problem: local motion is ambiguous because only the motion component perpendicular to a local edge can be determined from that region alone.

Global constraints resolve the aperture problem. The Horn-Schunck method imposes a global smoothness prior on the flow field. The Lucas-Kanade method assumes locally uniform flow within a small region, producing a well-conditioned system from the neighbourhood. Modern deep learning approaches — FlowNet, RAFT — learn to estimate flow from large datasets of synthetically generated ground truth, achieving state-of-the-art performance on standard benchmarks while inheriting their own failure modes under domain shift.

Dense optical flow estimates a motion vector for every pixel; sparse methods (feature tracking) track selected keypoints with high reliability. Applications include video stabilisation, action recognition, autonomous navigation, and visual odometry.

Sources: SRC-007. See source registry.

The Aperture Problem and Suspension

The aperture problem is the precise machine-vision instance of the focus integrity suspension challenge. A locally computed flow estimate is structurally underspecified — like a lens without its suspension system, it cannot determine its own position in the field without global context. Optical flow algorithms impose that context through regularisation, prior constraints, or learned global representations. When those constraints are violated by real-world motion, the flow estimate fails in a systematic, predictable way.

Relationship to the Governing Thesis

Optical flow is the mechanism by which machine vision systems maintain focus across time rather than within a single frame. The suspension analogy is precise: the zonules hold the lens in place against gravity and ciliary force; the flow field holds the scene representation in place against temporal disruption. Optical flow failure — like suspension failure — produces a cascade of downstream errors in any system that depends on temporal coherence.
