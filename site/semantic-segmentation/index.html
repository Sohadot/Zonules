Semantic Segmentation

Every pixel gets a label. Whether the label is correct is a question about interpretation, not about the image.

Definition

Semantic segmentation is the task of assigning a semantic class label to every pixel in an image, producing a complete partition of the image into contiguous labelled regions. It is the dense, pixel-level form of scene classification, distinguishing between categories such as road, sky, pedestrian, and vehicle without differentiating individual instances within a class.

Scientific Grounding

Semantic segmentation is typically implemented as a dense prediction problem using encoder-decoder architectures. The encoder produces a compressed, semantically rich feature representation; the decoder upsamples this to pixel resolution, assigning class probabilities to each location. Fully convolutional networks demonstrated that the same feature learning applied to classification could be extended to dense prediction by replacing fully connected layers with convolutional layers that preserve spatial information.

Instance segmentation extends semantic segmentation by additionally separating individual instances of the same class — distinguishing car-1 from car-2 rather than assigning all car pixels the same label. Panoptic segmentation unifies both tasks, requiring a complete, instance-aware partition of the image.

A systematic failure mode: models trained on limited class vocabularies assign confident, incorrect labels to out-of-vocabulary regions. A new class of object not in the training set will be assigned the most visually similar known class. This silent failure is a FIO-05 failure — the model's interpretation schema confidently produces incorrect output without signalling uncertainty.

Sources: SRC-007. See source registry.

Dense Labelling and Interpretation Scale

Semantic segmentation makes interpretation commitments at the pixel level rather than the image level. This scales the interpretation risk: a single image with 10 million pixels requires 10 million interpretation decisions, each of which can fail. At the same time, the local context at each pixel constrains the decision: neighbouring pixels in the same region are likely to share a class. This local coherence is both an asset (it can be exploited through spatial regularisation) and a vulnerability (spatially correlated errors can propagate across large regions).

Relationship to the Governing Thesis

Zonules.com treats semantic segmentation as the pixel-resolution instance of the interpretation challenge documented in L2 Visual Scene Parsing. Both partition a visual scene into labelled regions; both operate through interpretation mechanisms that can produce confident, structured, incorrect partitions. The Focus Integrity concern is the same: when a segmentation output is used as ground truth for downstream reasoning, its interpretation errors propagate.
