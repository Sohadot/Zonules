Size Constancy

The same object, the same size, however far it stands.

Definition

Size constancy is the tendency to perceive an object as having a stable physical size despite changes in the size of its retinal image with distance.

Scientific Grounding

The visual system achieves this by scaling the retinal image against perceived distance. When distance cues are removed or made misleading, size constancy breaks down and produces size illusions.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Size constancy is an accommodation-layer achievement — the percept holds steady as viewing conditions change.

Relationship to the Governing Thesis

Size Constancy is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-02 — Accommodation concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
