Small Object Detection

Can faint signal still be correctly read?

Definition

Small object detection addresses the difficulty of finding objects that occupy very few pixels.

Scientific Grounding

Small objects carry little feature information and are easily lost in downsampling. Higher input resolution and multi-scale features are common remedies.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Small object detection is an interpretation-layer limit: it tests whether faint signal can still be correctly read.
