Spatial Layout Perception

The stage on which objects are located.

Definition

Spatial layout perception is the apprehension of the overall geometric structure of a scene, such as its ground plane and boundaries.

Scientific Grounding

The layout can be grasped rapidly and guides navigation and the placement of objects. Surfaces such as the ground plane provide a reference frame for judging object positions and sizes.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Spatial layout perception is an interpretation-layer frame: it builds the stage on which objects are located.

Relationship to the Governing Thesis

Spatial Layout Perception is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result.
