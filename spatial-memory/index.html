Spatial Memory

Holding where things are when they leave view.

Definition

Spatial memory is the storage and retrieval of information about the locations of objects and the layout of environments.

Scientific Grounding

It supports navigation and the ability to return attention to remembered locations. It draws on brain regions including the hippocampus and parietal cortex.

*Sources: SRC-005, SRC-009. See the source registry for classification.*

Perceptual Role

Spatial memory is a suspension-layer map: it holds where things are when they are no longer in view.

Relationship to the Governing Thesis

Spatial Memory is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
