Synthetic Media

When appearance no longer proves origin, authenticity must be carried by an independent provenance chain.

Definition

Synthetic media encompasses outputs of generative systems including GAN-generated images, diffusion model outputs, NeRF renderings, and text-to-image synthesis.

Scientific Grounding

The indistinguishability of synthetic media from authentic capture constitutes a provenance integrity failure: origin can no longer be inferred from appearance alone. The C2PA standard provides cryptographically anchored metadata recording origin, edit history, and tool provenance, enabling authenticity verification independent of pixel analysis.

*Sources: SRC-007, SRC-008. See the source registry for classification.*

Machine-Vision Role

In the Zonules FIO framework, synthetic media is classified under FIO-04 Provenance: the structural challenge is that appearance alone cannot establish origin, requiring an independent provenance chain.

Relationship to the Governing Thesis

Synthetic Media is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-04 (Provenance) concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
