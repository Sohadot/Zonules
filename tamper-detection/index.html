Tamper Detection

Judging whether what is shown has been changed.

Definition

Tamper detection determines whether an image has been altered after capture.

Scientific Grounding

It combines cues such as inconsistent noise, lighting, and compression artifacts. Robust tamper detection must withstand benign edits like resizing while still catching malicious ones.

*Sources: SRC-023, SRC-024. See the source registry for classification.*

Machine-Vision Role

Tamper detection is a provenance-layer verdict: it judges whether what is shown has been changed.
