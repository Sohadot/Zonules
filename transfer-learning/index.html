Transfer Learning

Adapting a system's focus to a new operating range.

Definition

Transfer learning is the reuse of a model trained on one task or dataset to improve learning on a related task.

Scientific Grounding

Pretrained visual features are commonly adapted to new tasks with limited data. Transfer can fail when the source and target domains differ substantially.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Transfer learning is an accommodation mechanism (FIO-02): it adapts a system's focus to a new operating range.
