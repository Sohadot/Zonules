Video Object Segmentation

An object's identity held steady across time.

Definition

Video object segmentation tracks and delineates objects across the frames of a video.

Scientific Grounding

It must maintain a consistent object identity and mask despite motion, occlusion, and appearance change. Temporal coherence between frames is both a challenge and a source of information.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

Video object segmentation is a suspension-layer task: it holds an object's identity steady across time.
