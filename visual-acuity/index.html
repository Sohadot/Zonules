Visual Acuity

The upper bound on the detail a visual system can faithfully resolve.

Definition

Visual acuity is the capacity to resolve fine spatial detail, typically measured as the smallest discriminable feature.

Scientific Grounding

Acuity is highest at the fovea and falls rapidly with eccentricity into the periphery. Acuity depends jointly on optical quality, photoreceptor sampling density, and neural processing.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Visual acuity is a signal-fidelity measure (FIO-03): the upper bound on the detail the system can faithfully resolve.

Visual Acuity is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-03 — Signal-Noise concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
