Visual Crowding

Visual crowding is a signal-noise failure

Definition

Visual crowding is the impaired recognition of peripheral targets caused by the presence of nearby flanking elements.

Scientific Grounding

Crowding is a fundamental limit on peripheral vision: the critical spacing between target and flanker scales with eccentricity (Bouma's law). Crowding cannot be overcome by attention or practice and reflects a hard architectural constraint on how the visual system processes peripheral signals.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Visual crowding is a signal-noise failure (FIO-03): flankers degrade the target signal in the peripheral channel in a way that cannot be resolved by higher-level interpretation.

Relationship to the Governing Thesis

Visual Crowding is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-03 — Signal-Noise concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
