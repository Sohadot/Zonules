Visual Grounding

Linking a phrase to a region tests whether a model understands what language and image jointly specify.

Definition

Visual grounding requires identifying the image region referred to by a natural-language expression, linking linguistic semantics to spatial location in the image.

Scientific Grounding

Referring expression comprehension is the canonical visual grounding task, requiring a model to localize a unique referent described by a natural-language phrase. Visual grounding benchmarks reveal gaps in compositional understanding: models can succeed on individual words while failing on multi-attribute expressions requiring joint spatial and categorical reasoning.

*Sources: SRC-007. See the source registry for classification.*

Machine-Vision Role

In the Zonules FIO framework, visual grounding is classified under FIO-05 Interpretation: it requires the system to produce a correct interpretation of how language and image jointly specify a region.
