Visual Long-Term Memory

What we keep of an image long after it is gone.

Definition

Visual long-term memory is the durable store of visual information that can persist for hours, days, or years.

Scientific Grounding

Studies show people can remember thousands of images with surprising detail after a single viewing. Unlike working memory, its capacity appears very large and its contents relatively stable.

*Sources: SRC-005, SRC-009. See the source registry for classification.*

Perceptual Role

Visual long-term memory is a suspension-layer store: it holds visual structure available to perception long after the signal is gone.

Layer Classification

  • Layer: L2 — Perception

Relationship to the Governing Thesis

Visual Long-Term Memory is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
