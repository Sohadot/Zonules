Visual Memory Encoding

Deciding which of the seen becomes the remembered.

Definition

Visual memory encoding is the process of transforming a perceived image into a stored representation.

Scientific Grounding

Attention at the time of viewing strongly determines what is encoded and later remembered. Information encoded with meaning or organization is remembered better than unstructured detail.

*Sources: SRC-005, SRC-014. See the source registry for classification.*

Perceptual Role

Visual memory encoding is an interpretation-layer commitment: it decides which of the seen becomes the remembered.
