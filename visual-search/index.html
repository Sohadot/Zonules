Visual Search

Focus held and moved deliberately until the target is found.

Definition

Visual search is the task of locating a target among distractors in a visual scene.

Scientific Grounding

Search can be efficient and parallel or inefficient and serial, depending on target-distractor similarity. Feature-based search differs in difficulty from conjunction search, where targets are defined by combinations of features.

*Sources: SRC-005, SRC-006. See the source registry for classification.*

Perceptual Role

Visual search is a suspension of attention (FIO-01): focus is held and moved deliberately until the target is found.

Layer Classification

  • Layer: L2 — Perception
  • Cluster: attention
  • FIO class: FIO-01 — Suspension
  • FIS criterion: FIS-1

Relationship to the Governing Thesis

Visual Search is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as a FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
