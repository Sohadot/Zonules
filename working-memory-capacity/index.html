Working Memory Capacity

The few items focus can hold at once.

Definition

Working memory capacity is the limited number of items that can be held in an active, accessible state at once.

Scientific Grounding

Visual working memory is typically limited to only a few objects at a time. Capacity is constrained by both the number and the complexity of the items held.

*Sources: SRC-005, SRC-014. See the source registry for classification.*

Perceptual Role

Working memory capacity is a suspension-layer limit: it bounds how much focus can be held available at once.

  • Layer: L2 — Perception

Relationship to the Governing Thesis

Working Memory Capacity is one instance of the asset's governing claim: that focus is produced and held by structure that is not itself visible in the result. Its place in the focus-integrity ontology is fixed as an FIO-01 — Suspension concern, the same failure class whether the seeing system is made of tissue, attention, or silicon.
